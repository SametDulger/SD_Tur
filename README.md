# SDTur - Tur Operasyon Yönetim Sistemi


## 🛠️ Teknoloji Stack'i

- **Backend**: .NET 9, Entity Framework Core, JWT, Serilog
- **Frontend**: ASP.NET Core MVC, Bootstrap 5, Bootstrap Icons
- **Database**: SQL Server
- **Cache**: Redis
- **Monitoring**: Application Insights
- **Testing**: xUnit, Moq
